New Features Being Added

💳 Online Payment Gateway — SSLCommerz

Customers pay online using cards, bKash, Nagad, Rocket and internet banking — all through a single, trusted Bangladeshi gateway. Money settles directly to the business bank account. No monthly fee; the gateway charges a small percentage per successful sale only.

📱 WhatsApp Business Automation (Admin Alerts)

The website automatically sends the admin WhatsApp messages so the shop can be managed from the phone, without logging in. Alerts included:

  • Low-stock alert — when a product is about to run out
  • Back-in-stock update — when a product is restocked
  • New order alert — instantly when a customer places an order
  • Weekly sales summary — total orders & revenue every week
  • Monthly sales report — a bigger picture each month
  • Suggested extras — payment received alert, order cancelled alert, daily "good morning" sales snapshot, and best-selling product of the week
Note: The website is already built and working (storefront + admin panel + promotions + coupons). This proposal covers the full build plus the two new features above, priced item by item so you can see the value of each part.

WhatsApp Automation — Is It Free?

Short answer: the admin alerts are effectively free to run, but the right setup matters. There are two ways to do it — both are explained honestly below.

Option 1 — Official WhatsApp Cloud API ✔ Recommended

  • Cost: Meta gives a free monthly allowance (around 1,000 conversations/month). Since alerts only go to you (the admin), you almost always stay within the free tier. Beyond that, "utility" messages cost roughly ৳0.40–0.60 each — a few taka a month at most.
  • Reliability: Official and stable. Your number will not get banned.
  • What's needed: A free Meta Business account, a phone number for WhatsApp Business, and a one-time verification. The developer sets this all up.
  • Best for: A real business that wants alerts to always arrive. This is what the price includes.

Option 2 — Free Unofficial Method ⚠ Cheaper but risky

  • Cost: ৳0 — uses a free open-source tool that logs in like WhatsApp Web.
  • Risk: It is against WhatsApp's rules. The number can be blocked or banned without warning, and it can stop working after WhatsApp updates.
  • Upkeep: Needs the phone/session to stay connected; less stable.
  • Best for: Testing or a very tight budget where an occasional outage is acceptable.
Recommendation: Go with the Official Cloud API. For admin-only alerts the running cost is essentially free, and it protects your business number. The unofficial option is available on request if you prefer zero third-party setup.
